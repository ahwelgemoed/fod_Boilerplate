# Boiler Plate for FOD - CSS Animation

#### Project was created with ParcelsJS and SpectreCSS

##### How To Use

_Clone the Repo_

`$ git clone https://github.com/ahwelgemoed/fod_Boilerplate.git`

_cd to the folder_

`$ cd fod_boilerplate`

_Install Node_Modules_

`$ yarn` or `$ npm install`

_run the project_

`$ yarn start` or `$ npm run start`
